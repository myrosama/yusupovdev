DaemonClient

A secure, open-source cloud storage platform that provisions a user-owned Telegram bot and private channel to function as a personal, zero-cost file vault.

Overview

Unlike traditional cloud services that rent storage, DaemonClient gives users full ownership of their infrastructure. The platform provisions a Telegram bot and private channel for each user. The setup process consists of creating the bot, establishing a secure channel, and transferring ownership to the user. Once ownership is transferred, the developers delete their credentials, ensuring that only the user controls the vault.

How It Works

The protocol uses "atomic chunking"‚Äîlarge files are split into 19 MB encrypted shards in the browser to bypass Telegram's file-size limit and enable parallel uploads. A custom Cloudflare Worker acts as a proxy so that data flows from the user's device to Telegram without passing through DaemonClient's servers.

Security Model

The security model relies on AES-256-GCM encryption and PBKDF2 key derivation. Keys exist only in memory and are destroyed on logout. Files are encrypted before leaving the user's device, meaning that neither DaemonClient's servers nor Telegram can read the contents.
